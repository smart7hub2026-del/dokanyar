# دکان یار

پلتفرم مدیریت هوشمند کسب‌وکار و فروشگاه

## قابلیت‌های دکان یار

- مدیریت انبار و موجودی کالاها
- ثبت فاکتورهای فروش و خرید (نقد و نسیه)
- مدیریت مشتریان و بدهی‌های معوق
- گزارشات مالی و سود و زیان پیشرفته
- پشتیبانی از چاپ فاکتور در ابعاد مختلف
- ورود امن با حساب گوگل (Google OAuth 2.0)
- طراحی کاملاً ریسپانسیو برای موبایل و تبلت

## راهنمای تست و اجرا

### ۱. نصب وابستگی‌ها
```bash
npm install
```

### ۲. اجرای پروژه در حالت توسعه
```bash
npm run dev
```

### ۳. بررسی خطاهای تایپ‌اسکریپت
```bash
npm run typecheck
```

### ۴. ساخت نسخه نهایی (Production Build)
```bash
npm run build
```

### تست ورود با گوگل در موبایل
برای تست قابلیت ورود با گوگل در مرورگرهای موبایل:
1. پروژه را با `npm run dev -- --host` اجرا کنید.
2. با استفاده از IP محلی (مثلاً `http://192.168.1.10:5173`) در گوشی خود سایت را باز کنید.
3. دکمه "ورود با Google" را در صفحه لاگین کلیک کنید.
4. فرآیند احراز هویت باید به درستی و بدون خطای دستگاه انجام شود.
